важные моменты.

Код сервера:
https://github.com/4uck/myProjectServer.git




Идея:
Приложения для инвентаризации. Имеется база данных, в ней хранится инвентарь
вместе с инвентарным штрихкодом и т.д. Требуется провести инвентаризацию 
кабинета №137. На первом экране приложения выбираем его, далее проходим по
 всему инвентарю этого кабинета, сканируем штрихкоды(кнопка "Scan"), после по нажатию кнопки "GetReport" на сервере
скрипт пишет отчет в файл "test.xlsx" на наличие или отсутствие инвентаря.


База данных: на сервере имеется база данных "test". В ней две таблицы "first" и "room".

структура "first":
id (int)
name (text)
code (text)
room_id (int) id кабинета из таблицы room

структура "room":
id (int)
name (text)


Работает в пределах одной локальной сети (ip сервера: 192.168.1.40).
